## Add to project

```sh
yarn add nde-design-system
```

_or_

```sh
npm i nde-design-system --save
```

---

## Usage

- Components require a Theme Provider to apply the styles.

```jsx
import {ThemeProvider} from 'nde-ui';

<ThemeProvider>
  <App />
</ThemeProvider>;
```

To view all components available, `npm run storybook`.
